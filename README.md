# WMSVUE

## Project setup
```
yarn
```

### Compiles and hot-reloads for development
```
yarn start
```

### Compiles and minifies for production
```
yarn build
```

### Run your tests (Jest + Cypress)
```
yarn test
```

### Run your unit tests
```
yarn unit
```

### Run your end-to-end tests
```
yarn e2e
```

### Run your end-to-end in UI tests
```
yarn e2e:open
```

### Lints and fixes files
```
yarn lint
```
